Requirement: Share price confirmation on transaction completion
Status: Parked — to be built as part of deal workflow redesign
When a buy or sell transaction is completed, the user should be prompted to confirm the share price update before the deal closes. The prompt should say something like: "This transaction is updating the share price to £[transaction price]. Is this correct? If not, should the price remain at £[current price] or would you like to enter a new price manually?"
Applies to: all deal types (buy and sell) except CLN conversions which are out of scope.
Backfill: not required — existing data is test data.
Dependency: requires deal workflow redesign to be completed first.
